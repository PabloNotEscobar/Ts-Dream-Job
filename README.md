# Our online courses

Карточный каталог курсов с живым поиском и фильтрацией по категориям. Свёрстан по макету из Figma.

**Демо:** _ссылка на Vercel_
**Репозиторий:** _ссылка на GitHub_

## Стек

- HTML
- SCSS (модульная структура, методология BEM)
- Vanilla JavaScript — без библиотек и фреймворков

## Возможности

- Pixel-perfect вёрстка по макету
- Резиновый адаптив от 1920 до 320 px без жёстких брейкпоинтов: типографика и отступы масштабируются через `clamp()`, сетка карточек перестраивается через `auto-fill` + `minmax`
- Живой поиск по названию курса (без перезагрузки)
- Фильтрация по категориям с подсчётом количества курсов в каждой
- Кнопка «Load more» с порционной подгрузкой

## Структура

```
online-courses/
├── index.html
├── css/
│   └── style.css          # собирается из SCSS
├── js/
│   ├── courses.js         # данные курсов и категорий
│   └── app.js             # рендер, поиск, фильтрация, подгрузка
└── src/
    └── scss/
        ├── main.scss
        ├── _variables.scss
        ├── _mixins.scss   # функция fluid() для резинового масштабирования
        ├── _base.scss
        └── blocks/        # по одному файлу на BEM-блок
            ├── _courses.scss
            ├── _tabs.scss
            ├── _search.scss
            ├── _card.scss
            └── _load-more.scss
```

## Запуск локально

```bash
npm install
npm run build      # собрать CSS из SCSS
npm run watch      # пересборка при изменениях
```

Открыть `index.html` через любой статический сервер, например:

```bash
npx serve .
```
